# Video Learning Helper - 用户注册和登录功能演示

## 🎯 演示目标
展示AI拉片助手的用户注册和登录功能，验证与Supabase数据库的完整集成。

## 🚀 演示环境
- **前端**: http://localhost:3000 (Next.js + Tailwind CSS)
- **后端**: http://localhost:8000 (FastAPI + Supabase)
- **数据库**: Supabase PostgreSQL (已有10个测试用户)

## 📋 演示步骤

### 1. 打开应用首页
1. 在浏览器中访问: `http://localhost:3000`
2. 查看AI拉片助手的主页界面
3. 注意页面的现代化设计和响应式布局

### 2. 演示用户注册功能

#### 步骤 2.1: 访问注册页面
1. 点击页面右上角的"注册"按钮
2. 或直接访问: `http://localhost:3000/register`

#### 步骤 2.2: 填写注册信息
**测试用户信息:**
```
姓名: 演示用户
邮箱: demo@example.com
密码: demo123456
确认密码: demo123456
```

#### 步骤 2.3: 提交注册
1. 点击"注册"按钮
2. 观察加载状态和成功提示
3. 注意成功后自动跳转到登录页面

### 3. 演示用户登录功能

#### 步骤 3.1: 使用新注册的账户登录
**登录信息:**
```
邮箱: demo@example.com
密码: demo123456
```

#### 步骤 3.2: 演示现有用户登录
**可选择以下任一测试账户:**
```
管理员账户:
邮箱: admin@example.com
密码: admin123456

剪辑师账户:
邮箱: editor1@example.com  
密码: editor123456

教师账户:
邮箱: teacher@example.com
密码: teacher123456
```

### 4. 验证登录状态
1. 登录成功后查看页面变化
2. 检查用户信息是否正确显示
3. 验证JWT token是否正确保存

### 5. 演示错误处理

#### 5.1: 重复注册测试
1. 尝试使用已存在的邮箱注册
2. 观察错误提示: "该邮箱已被注册"

#### 5.2: 错误密码测试
1. 使用正确邮箱但错误密码登录
2. 观察错误提示: "邮箱或密码错误"

#### 5.3: 格式验证测试
1. 输入无效邮箱格式
2. 输入过短密码
3. 观察实时验证提示
